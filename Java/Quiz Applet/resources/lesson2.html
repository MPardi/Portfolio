Intro to function combination

Adding, subtracting, multipying, and dividing functions is possible just as we can add, subtract, multiply and divide numbers.

Function addition

Creating a function out of the addition of two functions

Let's add f(x) = x + 2 and g(x) = 3x to make a new function.

    h(x) = f(x) + g(x) = (x + 2) + (3x)

    h(x) = x + 2 + 3x

    h(x) = 4x + 2

We now have our new function h(x) = 4x + 2.

Evaluating a combined function

It is also possible to evaluate combined functions for particular inputs. Let's evaluate h(x) = f(x) + g(x) = 4x + 2 at x = 4. You are able to solve this using one of two possible ways.

Method A: Finding f(4) and g(4), then adding the results.

    f(x) = x + 2

    f(4) = (4) + 2

    f(4) = 6


    g(x) = 3x

    g(4) = 3(4)

    g(4) = 12

    Now by adding f(4) and g(4) you get h(4).

    h(4) = f(4) + g(4)

    h(4) = 6 + 12

    h(4) = 18

    h(4) = 18

Method B: Subtituting the given value of x = 4 into the previously calculated h(x) = x + 2 function.

    h(x) = 4x + 2

    h(4) = 4(4) + 2

    = 16 + 2

    h(4) = 18

As you can see, both method return the same result in the end. You should feel free to try both methods and see which one you are more comfortable with.


Function subtraction

Creating a function out of the subtraction of two functions

Let's subtract f(x) = 7x + 4 and g(x) = 8x - 3 to make a new function h(x).

    h(x) = f(x) - g(x)

    h(x) = (7x + 4) - (8x - 3)

    h(x) = 7x + 4 - 8x + 3

    h(x) = -x + 7

We now have our new function h(x) = -x + 7.

Evaluating h(x)

Using the same two methods shown above, we can also find the value of h(x) = f(x) - g(x) at x = 12.

Method A:Finding the difference between the results of f(12) and g(12).

    f(x) = 7x + 4

    f(12) = 7(12) + 4

    f(12) = 84 + 4

    f(12) = 88


    g(x) = 8x - 3

    g(12) = 8(12) - 3

    g(12) = 96 - 3

    g(12) = 93

    Now by subtracting g(12) from f(12) we get h(12).

    h(12) = f(12) - g(12)

    h(12) = 88 - 93

    h(12) = -5

Method B: Subtituting the given value of x = 12 into h(x) = -x + 7.

    h(x) = -x + 7

    h(12) = -(12) + 7

    h(12) = -5

As you can see, both Method A and Method B gave us the same result of h(x) = -5. In this case, it is apparent that Method B would take much less time.


Function multiplication

Creating a function out of the multiplucation of two functions

Let's multiply f(x) = 2x + 1 and g(x) = 3x to make a new function.

    h(x) = f(x) · g(x) = (2x + 1) · (3x)

    h(x) = 2x · 3x + 1 · 3x

    h(x) = 6x2 + 3x

We now have our new function h(x) = 6xy + 3y through simple distribution.

Evaluating h(x)

It is also possible to evalue the combined function h(x) like we did with addition. Let's say we're evaluating h(x) = f(x) · g(x) at x = 6. Using the same 2 methods we can evaluate h(x)

Method A: Finding f(6) and g(6), then multiplying the results.

    f(x) = 2x + 1

    f(6) = 2(6) + 1

    f(6) = 12 + 1

    f(6) = 13


    g(x) = 3x

    g(6) = 3(6)

    g(6) = 18

    Now by multiplying the results of f(6) and we get h(4).

    h(6) = f(6) · g(6)

    h(6) = 13 · 18

    h(6) = 234

Method B: Subtituting the given value of x = 6 into the previously calculated h(x) = 6x2+3x function.

    h(x) = 6x2 + 3x

    h(6) = 6(6)2 + 3(6)

    h(6) = 216 + 18

    h(6) = 234

As you can see, both Method A and Method B yielded the same results: h(6) = 234.
